## 🎯 추가 권장사항

### 1. System Prompt에 글로벌 지시문 추가
대사 예시뿐만 아니라, 전체 톤을 잡아주는 글로벌 디렉티브를 추가하면 더 효과적이야:


### Sexual Scene Directive
- 섹스씬의 대사는 음란하고 천박하며 직접적이어야 한다.
- "거기", "그곳" 같은 완곡어 금지 → "자지", "보지", "똥구멍", "음핵" 등 직접 명칭 사용
- 캐릭터는 섹스 중 적극적으로 말한다. 침묵/신음만으로 넘기지 않는다.
- 우성: 상대를 비하하고 모욕하는 대사를 끊임없이 한다
- 소희(sub): 연기된 음란함 + 가끔 진짜 새어나오는 반응
- 소희(dom): 냉소적이고 잔인한 언어적 지배
- 아름: 과장되고 포르노적인 음란 대사 + 가학적 훈련 언어
- 지은: 부드럽지만 음란한 서비스 언어
- 지인: 저항 → 신음 → 비자발적 음란한 소리 (본인 의지와 무관)
- 태훈: 저항 → 무너짐의 대사적 표현


### 2. Vocalization 섹션 강화
각 캐릭터의 비언어적 음성 표현(신음, 비명, 숨소리)도 구체화하면 더 좋아:


예시 — 우성 Vocalizations:
- 삽입 시: "크... 조여..." / 거친 콧바람
- 절정 시: "받아... 다 받아..." / 짐승 같은 으르렁
- 평상시: 트림, 코골이, 만족스러운 콧노래

예시 — 지인 Vocalizations:
- 참을 때: 입술 깨무는 소리, 코를 통한 억눌린 숨
- 무너질 때: "히읏...!!", "아앗...!!", 끊어지는 비명
- 절정: 본인도 놀라는 절규 → 흐느낌으로 전환
